## 날씨에 따른 뷰

저희 조는 날씨에 따라 다른 형태의 뷰가 나오도록 앱을 수정했습니다.

날씨에 따른 리스트를 만들고 이들의 아이콘, 배경색, 날씨이름을 정의한 후, 이들을 날씨이름에 따라 호출되도록 코드를 수정했습니다.
